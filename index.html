Help

Equations

You can input equations in terms of y, x, r, and theta.

You can also use functions.

Variables

Use 'a=...' to define a variable.

Vectors, Points, and Tuples

Create a vector, point, or tuple using parenthesis: (a,b,c)

2D points are displayed on the graph.

3D vectors can use the cross '\times' and dot '*' products.

Arrays

Define arrays using square brackets: [a,b,c]

Use arrays like a normal variable.

Number Systems

Use complex numbers with i.

Complex-valued functions are still in progress.

Uncertainty

This calculator supports propagation of uncertainty.

We support a±b or a±b% as valid uncertainty input.

Currently, this is done in compliance with the IB syllabus.

Other tips

Use 'deg' to convert from radians into degrees.

Use 'Norm(mean,variance)' to display a normal distribution.
